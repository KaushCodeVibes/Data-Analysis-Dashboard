# Netflix Data Analysis Dashboard

A professional data analytics project focused on understanding Netflix movie trends using data preprocessing, data cleaning, and interactive data visualization. This dashboard is designed to support data‑driven business decisions for content strategy and audience insights.

---

## Project Overview

Netflix is widely recognized for its work in data science, artificial intelligence (AI), and machine learning (ML), especially in building recommendation systems based on user behavior and viewing patterns.

In this project, a dataset containing information on more than **9,000 Netflix movies** is analyzed to extract meaningful insights and present them through an interactive dashboard.

The complete workflow includes:

* Data preprocessing
* Data cleaning
* Exploratory data analysis (EDA)
* Interactive data visualization

---

## Business Problem

The objective of this project is to help stakeholders understand:

* content distribution across genres,
* popularity trends,
* voting patterns, and
* yearly release behavior on Netflix.

These insights can support strategic decisions related to content acquisition, production planning, and platform optimization.

---

## Key Analysis Questions

1. Which genre is most frequently released on Netflix?
2. Which movie genre has received the highest number of votes overall?
3. Which movie has the highest popularity score and what is its genre?
4. Which movie has the lowest popularity score and what is its genre?
5. Which year recorded the highest number of movie releases on Netflix?
